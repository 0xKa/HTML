<head> tag

Concept:

The <head> element does not render any content directly but contains metadata that instructs the browser on how to handle the rest of the HTML document. It includes elements that can affect the entire page or set specific instructions for the browser and search engines.

Key Components of the <head>:

  • Title Tag: Specifies the title of the document, which appears in the browser title bar or tab.
  • Meta Tags: Provide essential metadata like character set, page description, keywords, author of the document, and viewport settings.
  • Link Tag: Used to link external resources such as CSS files and favicons.
  • Script Tag: Place to include JavaScript files or code that affects the HTML document.
  • Style Tag: Contains CSS styles that are applied to the HTML document.

Favicons:

  • Purpose of Favicons
    • Favicons serve as a visual marker for websites, aiding users in identifying and quickly locating a website's tab among multiple open tabs. They are crucial for branding and usability.
  • Types of Favicons
    • Classic Favicon: Typically a 16x16 pixel image, displayed in tabs and bookmarks.
    • Touch Icons: Used for Apple and Android devices when the site is saved to the home screen.
    • SVG Icons: Scalable icons that are increasingly used for their flexibility and resolution independence.

<meta> tag

Metadata provides essential information about a webpage, such as content description, authorship, and instructions for browsers on how to display content.
