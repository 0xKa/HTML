Unorderd Lists

        - In HTML, an unordered list is created with the <ul> tag.
        - Each item within the list is marked up with an <li> (list item) tag.
        - The default visual representation of an unordered list is a bullet point for each item.
Example1
  • item1
  • item2
  • item3
Example2
  • item1
    • item1.1
    • item1.2
    • item1.3
  • item2
    • item2.1
    • item2.2
      • item2.2.1
    • item2.3
  • item3
      
Orderd Lists

        - Ordered List: The <ol> tag is used to define an ordered list in HTML.
        - Items within the list are automatically numbered by the browser,
             which helps in indicating a clear order or sequence.
        - The numbering can be customized with different styles such as numeric,
             alphabetical, or Roman numerals.
Example1
  1. item1
    • item1.1
    • item1.2
  2. item2
    1. item2.1
    2. github/0xKa
    3. item2.3
    4. item2.4
  3. item3
Example2
  1. item1
  2. item2
  3. item3
    
Description Lists

    - Description List: The <dl> tag in HTML defines a description list.
    - Unlike ordered (<ol>) and unordered (<ul>) lists, 
        description lists consist of a series of terms and their corresponding descriptions.
    - Components:
        <dt> (Description Term): This tag is used to indicate the term being described.
        <dd> (Description Details): This tag provides the details or the description of the term.
